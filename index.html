Handcrafted Design System

A minimal baseline stylesheet maintained by Handcrafted Technologies, the tech division of Archangel. Covers settings, reset, typography, and element defaults - the base layer for you to customise with less override and more extensibility.

  • Semantic CSS custom properties (--color-*, --font-*)
  • Mona Sans & Hubot Sans variable fonts
  • Light and dark mode via prefers-color-scheme
  • Two entry points: standard and Tailwind v4

Standard usage

Link in your head element:

<link rel="stylesheet" href="https://static.wearearchangel.com/styles/main.css">

Or import in your CSS:

@import url('https://static.wearearchangel.com/styles/main.css');

Tailwind v4 usage

Use main.tailwind.css as your CSS entry point passed to the Tailwind CLI or Vite plugin. It imports Tailwind, registers all design system tokens in @theme, and restores the brand baseline in @layer base:

@import url('https://static.wearearchangel.com/styles/main.tailwind.css');

Then extend with your own @theme tokens and @layer components rules.

Text

Typeface

Body text uses Mona Sans (--font-sans) - a variable sans-serif for UI and prose. Headings use Hubot Sans (--font-display) - a variable sans-serif with a tighter, more expressive stance. Both fall back to the system sans-serif stack.

Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

[Top]

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.
